关于我们的小组

组长： 李哲彦
组员： 华圣佳 邹品聪 连亮瑜

我们小组的作品主要是一个图片修复生成器网站，作品分为前端和后端服务器。前端负责展示用户界面并收发用户图片，后端负责保存并处理图片，同时也保存前端的底层逻辑代码。
前端服务器的网页使用了AI Doodle的开源模型，包括各个网页的html和php模板，在此基础上修改完成。
后端我们的源程序main.py使用了GFPGAN,Real-ESRGAN,Sharpen,Denoise,Colorize,Api等模型，借助这些模型我们才能够完成图片修复的工作，感谢这些模型的作者为我们提供了如此便利的工具。

我们使用的模型


我们的后端处理程序借助使用网上发布的模型，特此鸣谢这些模型的作者

01
Sharpen

Github上的开源模型，使用一定的算法对图像进行处理以检测图像中存在的边缘，然后对符合条件的边缘采取一定的滤波措施以提高边缘锐度。
感谢作者ironcev，afrlan，shankyjain7243，tom-englert



Learn More on Github
02
DDColor

DDColor可以为历史黑白老照片提供生动自然的色彩

感谢作者Xiaoyang Kang, Tao Yang, Wenqi Ouyang, Peiran Ren, Lingzhi Li, Xuansong Xie
DAMO Academy, Alibaba Group



Learn More on Github Learn More on arxiv
03
GFPGAN

GFPGAN旨在开发一种用于真实世界人脸恢复的实用算法。 它利用封装在预训练人脸GAN（例如StyleGAN2）中的丰富多样的先验进行盲脸恢复。

感谢作者Xintao Wang, Yu Li, Honglun Zhang, Ying Shan Applied Research Center(ARC), Tencent PCG



Learn More on Github
04
Real-ESRGAN

增强的超分辨率的对抗生成网络，是由腾讯ARC实验室发布的一个盲图像超分辨率模型，它的目标是开发出实用的图像修复算法，感谢作者Xintao Wang, Liangbin Xie, Chao Dong, Ying Shan



Learn More on Github Learn More on arxiv

Denoise

图像去噪，用于平滑图像并保留细节。感谢作者adefossez，adiyoss，hoyaaaa，wesbz
